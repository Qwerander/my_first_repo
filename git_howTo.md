# Команды по гиту

Создание репозитория
```sh
git init 
```

Посмотреть статус
```sh
git status
```

Добавить в изменения файл
```sh
git add <filename>
```

Добавить коммит 
```sh
git commit -m "сообщение коммита"
```

Посмотреть лог
```sh
git log 
```

Посмотреть лог в одну строку
```sh
git log --oneline
```
